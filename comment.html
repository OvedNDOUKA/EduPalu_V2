C'est fait comment ?

EduPalu est une application d'information et d'éducation sur le paludisme au Congo.

Cette application utilise les technologies web les plus récentes : HTML5, CSS3 et JavaScript.

EduPalu est un logiciel libre, que vous pouvez utiliser, modifier, copier comme bon vous semble.
Le code source d'EduPalu est disponible sur le site de développement collaboratif GitHub. Vous pouvez le télécharger à cette adresse :
http://github.com/fongwama/EduPalu

Le test de diagnostic rapide

Pour faire un test de diagnostic rapide (TDR), le technicien doit prélever une goutte de sang. Cette goutte est ensuite déposée sur une bande de papier spécial qui au bout de 15 à 20 min indiquera la présence ou non du parasite. Le TDR est très rapide mais il ne permet pas de connaître le nombre de parasites présents dans le sang.

Vous pouvez trouver des tests de diagnostic rapide en pharmacie et faire le test vous-même à la maison. Par contre, seul un médecin pourra prescrire le bon traitement si vous avez le paludisme. Alors allez le voir sans tarder avec votre test.
